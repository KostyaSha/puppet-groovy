Groovy Puppet Module
======

This module will download and setup groovy from the official Codehaus' repository.

It works for the following linux distributions:

* Ubuntu
* Fedory
* Debian

Usage
-----

```puppet
class {'groovy':
  version => '2.1.3'
}
```
